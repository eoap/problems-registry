    ProblemDetailsRegistry (1.0.0)

    Download OpenAPI specification:

    URL: https://eoap.github.io/problems-registry License: Apache 2.0

    The EOAP Problem Details Registry acts as a well known location for problem detail types returned by various SmartBear APIs. It acts as a central registry to promote reuse and clarity of problem detail types conforming to RFC9457 (formerly RFC7807).
